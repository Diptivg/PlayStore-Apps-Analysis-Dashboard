##  1. Project Overview  

The **Google Play Store Apps Analysis Dashboard** provides valuable insights into app performance, popularity, and user engagement across various categories.  
This project helps **developers, marketers, and analysts** understand market trends, improve app strategies, and make data-driven decisions for better app success.  

---

##  2. Purpose  

To analyze Google Play Store app data ‚Äî including **installs, ratings, reviews, and pricing** ‚Äî and visualize meaningful patterns that highlight the **most successful app types** and **user preferences**.  

---

##  3. Tech Stack  

-  **Power BI** ‚Äì For creating interactive data visualizations and dashboards  
-  **Power Query** ‚Äì For data cleaning and transformation  
-  **DAX (Data Analysis Expressions)** ‚Äì For creating custom measures and calculated columns  
-  **Microsoft Excel / CSV** ‚Äì Used as the data source for import and preprocessing  

---

##  4. Data Source  

- **Source:** Soft Byte ‚Äì Google Play Store Dataset  

**Structure Includes:**  
-  **App Name:** Name of the Android application  
-  **Category:** Type or classification (e.g., Tools, Games, Education)  
-  **Rating:** Average user rating  
-  **Reviews:** Total number of user reviews  
-  **Installs:** Number of downloads  
-  **Size:** App size in MB  
-  **Type:** Free or Paid  
-  **Price:** Price of the app (if paid)  
-  **Rating:** Age group suitability  
-  **Genres:** App subcategory  
-  **Last Updated:** Last updated date  

---

##  5. Dashboard Features & Highlights  

- **Category-wise Analysis -:** Explore which app categories dominate the Play Store by number of apps and total installs.  
- **User Ratings Overview -:** Understand user satisfaction levels through average rating distribution.  
- **Install Trends -:** Compare installs across free vs. paid apps and top-performing categories.  
- **Price Impact Visualization -:** Analyze how pricing affects app popularity and user ratings.  
- **Update Frequency Insights -:** Track how often apps are updated to measure developer engagement.  
- **Dynamic Filters -:** Interact with data by selecting categories, content ratings, or app types.  
- **Key KPIs Displayed:**  
  - Total Apps  
  - Average Rating  
  - Total Installs  
  - Top Categories  
  - Most Reviewed Apps  

---

##  6. Key Insights  

- *Games* and *Tools* categories dominate in installs and availability.  
- *Free apps* receive far higher install volumes than paid apps.  
-  Apps that are updated more frequently tend to have **higher average ratings**.  
-  The *Education* category shows steady growth in both ratings and installs.

##  8. How to Use  

1. Open the `.pbix` file in **Microsoft Power BI Desktop**.  
2. Navigate between dashboard pages using the tabs at the bottom.  
3. Use **filters and slicers** to interact with the data dynamically.  
4. Explore category-wise, rating-wise, and type-wise visualizations for detailed insights.
